Tipos de errores en C++

Error lógico

Un error lógico es un error en el programa código fuente eso da como resultado un comportamiento incorrecto o inesperado. Es un tipo de Error de tiempo de ejecución eso simplemente puede producir el mal salida o puede hacer que un programa se bloquee mientras se ejecuta.

Error taxonómico

Un error taxonómico o de clasificación de especies acentúa los riesgos de extinción de una variedad específica de raya, informó el Centro de Investigaciones Científicas de Francia.
